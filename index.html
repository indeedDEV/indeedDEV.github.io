BERET

A Discord Bot for all of your Discord needs

Setup

1. Go to the invite link.
2. Select the Discord server you would like to invite BERET to from the dropdown menu.
3. If you give BERET the Administrator permission you're all set to use BERET.
If not, select the permissions you want to give to BERET in Discord. Not having certain permissions will limit its functionality.
4. Enjoy your improved Discord experience!


Minecraft Resource Packs

indeed's GUI Edit (or iGE) enhances the minecraft GUI with transparent item backgrounds for a less obstructive GUI.
indeed's Lunar-Sol Edit (or iLSE) makes visual changes to the sun and moon for a more realistic and appealing sky.

• indeed's GUI Edit -- compatible with Minecraft Version 1.16.5 -- Download
• indeed's Lunar-Sol Edit -- compatible with Minecraft Version 1.16.5 -- Download

Setup

1. Download the Resource Pack you want to use.
2. Open Minecraft and go to Options... > Resource Packs... > Open Pack Folder.
3. Move or copy the Resource Pack that you downloaded to the Pack Folder. The Resource Pack should now appear under Available in Minecraft's Resource Packs Menu.
4. Move the Resource Pack from Available to Selected.
5. Enjoy your new Minecraft experience!

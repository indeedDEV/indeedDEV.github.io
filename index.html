BERET

A Discord Bot for all of your Discord needs.

1. Go to https://discord.com/oauth2/authorize?client_id=807641111140499466&scope=bot&permissions=8
2. Select the Discord server you would like to invite BERET to from the dropdown menu.
3. If you give BERET 'administrator' permissions, you're all set to use BERET.
If not, select the permissions you want to give to BERET in Discord. Not having certain permissions will limit its functionality.
4. Enjoy your improved Discord experience!


indeed's Gui Edit

A Minecraft Resource Pack for an Enhanced GUI.
Compatible with Minecraft version 1.16.5

1. Download iGE.zip
2. Open Minecraft and go to Options... > Resource Packs... > Open Pack Folder.
3. Move or copy iGE.zip to the Pack Folder. iGE.zip should now appear under 'Available' in Minecraft's Resource Packs Menu.
4. Move iGE.zip from 'Available' to 'Selected'.
5. Enjoy your new Minecraft experience!
